# 国标麻将

与雀魂的立直麻将相比: 国标麻将**8番起和**, 有头跳, 无振听, 这里实现的国标没有花牌

以下番种主要根据雀渣平台规则(明暗杠设为5番, 但不计天地人和)

对于诈和及不足8番的错和, 这里按照赔每家各 2400 点计算
